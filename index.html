AI Agent (Google Colab)

An AI agent built in Python that answers questions about cosmetic product ingredients, their safety, and their effect on the skin. It uses a Retrieval-Augmented Generation (RAG) approach, grounding answers in trusted sources — the official Serbian cosmetics regulation, professional literature, and local natural-cosmetics brands — instead of relying solely on the model's built-in knowledge. The system combines a Chroma vector database with multilingual BAAI/bge-m3 embeddings, a Groq-hosted Llama 3.1 model, and Tavily web search, all orchestrated with LangGraph. A router node autonomously decides whether each question is answered directly, from the local knowledge base, or via live web search, and the agent keeps conversation memory across turns.
